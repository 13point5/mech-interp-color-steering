Introduction

Welcome to DevDocs, a comprehensive documentation site for developers. This site is designed to help you understand, use, and contribute to our open-source project.

Installation

To get started, simply include the following script in your HTML file:

<script src="https://cdn.jsdelivr.net/npm/devdocs@latest/dist/devdocs.min.js"></script>

Or install via npm:

npm install devdocs

Usage

Once installed, you can use the DevDocs library to create interactive documentation for your projects. Here's a simple example:

<div id="doc"></div> <script> const doc = new DevDocs(); doc.init('#doc'); </script>

API

The DevDocs library provides a simple API to customize your documentation:

  • new DevDocs() - Initialize the documentation
  • doc.init(container) - Render the documentation in a container
  • doc.setTheme(theme) - Set the theme for the documentation

Contribution

We welcome contributions to this project. If you'd like to help, please fork the repository and submit a pull request.
